AI Agent Design & Agent Flow 과정

발표자: 전대호 MVP, 진미나 MVP

Microsoft Copilot을 활용한 디자인 씽킹부터 Copilot Studio의 Agent Flow까지, 초보자도 쉽게 따라할 수 있는 실습 중심 세션입니다. 개인의 업무 스타일에 맞는 AI 어시스턴트를 구상하고, 실제로 일을 대신해주는 AI를 직접 만들어봅니다.

📚 주요 실습 내용

✅ 디자인 씽킹 이해

✅ 업무 문제 정의 및 AI 역할 설계

✅ 개인 맞춤 AI 부사수 컨셉 완성

✅ Copilot Studio & Agent Flow 이해

✅ Agent Flow로 일하는 AI 만들기

✅ 업무 자동화 시나리오 완성

Microsoft Copilot Studio 기반
에이전트 마스터 과정

📚 주요 실습 내용

✅ 개발 환경 설정

✅ Copilot Studio 입문

✅ 다양한 기능 사용

✅ 여러 도구 추가하기

✅ SharePoint 연동

✅ 나만의 에이전트 만들기
